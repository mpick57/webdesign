installation

Server Requirements

  • PHP >= 7.1.5
  • Tokenizer PHP Extension
  • XML PHP Extension
  • Ctype PHP Extension
  • JSON PHP Extension
  • OpenSSL PHP Extension
  • GD PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • BCMath PHP Extension
  • Exif PHP Extension

Database requirements

  • If you using MySQL you have to use version older than the 5.7.7

Step by step installation

  1. Extract downloaded archive
  2. Upload all files from Upload folder to the server
  3. Directories storage, bootstrap (CHMOD 0777) chmod -R 777 storage/ bootstrap/ .env.example
  4. Rename .env.example to .env
  5. Go to web url: http://your-web.site/install
  6. Follow instructions on the screen

Use App

Setup Email

You will need to configure SMTP settings for Saas Resume Builder if you want to send e.q. register, SubscriptionExpired, SubscriptionPaid.

Scheduling

To start watching send mail and check SubscriptionExpired you only need to add the following Cron entry to your server: .

* * * * * /usr/bin/php /path-to-your-project/artisan schedule:run >> /dev/null 2>&1

Setup Payment

You will need to configure Payment settings for Saas Resume Builer.

  • Paypal: https://developer.paypal.com/docs/api/overview
  • Stripe: https://stripe.com/docs/keys

Setup Currencies

Add new currencies.

    2.

                  'name'     => 'Language name',

                ],
  2. Copy folder /resources/lang/en/ to /resources/lang/xx/
  3. Copy file /resources/lang/pt.json to /resources/lang/xx.json
  4. Make necessary translations
